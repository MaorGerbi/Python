Write a program that catches the player:
A- string representing a path to a text file.
  The text file will contain a space-separated list of words.
B- An integer representing the location of a particular word in a file.
   The word in the recorded location will be the secret word the player will have to guess.

From this moment on, the game will continue according to the rules of "dependent man" -
 at each stage the player will guess, receive feedback according to the presence of the letter 
in the secret word and according to his guesses so far.
